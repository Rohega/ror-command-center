# Docs Directory

Project documentation for RoR Command Center and consuming applications.

Start here as a human: [docs/README.md](README.md) (three doors: use, contribute, example).

## Structure

| Path | Purpose |
|------|---------|
| `docs/README.md` | Documentation map — install / contribute / warehouse example |
| `docs/USER-MANUAL.md` | Human onboarding hub (Path A / Ollama / Docker) |
| `docs/INSTALL.md` | Path A detail (`install.sh` into an existing project) |
| `docs/rorcc-cli.md` | `rorcc` CLI reference (local Ollama / cloud) |
| `docs/how-to/` | Task-oriented how-tos (agents, workflows, create specialist) |
| `docs/integrations/` | Per-platform AI setup (Cursor, Claude, Codex, …) |
| `docs/COLLABORATIVE-DESIGN-PRINCIPLE.md` | Long examples of the collaboration protocol |
| `docs/architecture/` | ADRs (app-owned; warehouse ADRs here are EXAMPLE ONLY) |
| `docs/specs/` | Feature specifications (warehouse-mvp is EXAMPLE ONLY) |
| `docs/stories/` | User stories (warehouse-mvp is EXAMPLE ONLY) |
| `docs/modules/` | Module documentation (empty in the kit; filled in apps) |
| `docs/runbooks/` | Operational runbooks |
